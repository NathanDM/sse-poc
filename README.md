## SSE server

Simple SSE server.

- [ ] Gulp task
- [X] REST API To manage CRUD
- [X] Electron panel to easily send Server Event
- [ ] Unit test


## Dev

### Install Dependencies

```sh
npm install
```

### Launch the App with Watcher (js, css, html, electron)

```sh
npm run watchify
```

### Launch server with panel
````sh
npm start
```
